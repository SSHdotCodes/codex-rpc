Three moods, tracked live

codex-rpc tails Codex's session logs locally — nothing leaves your machine — and knows a running command from a thinking model.

🧠 Thinkingreasoning, planning, digesting command output
⌨️ Workingediting files, searching, running builds & tests
😴 Sleepingidle, task complete, or Codex not running

Why it's neat

Truly live statesUnderstands Codex's event log: a 10-minute build shows ⚙️ the whole time; quiet after results means 🧠.
Lifetime token counterSums every session you've ever run and ticks up live, right on your profile.
Zero dependenciesOne Node file. No npm installs, no Electron, no telemetry. Talks straight to Discord's local socket.
Runs itselfInstalls as a background agent — starts at login, restarts if it crashes, no terminal window.
Seamless loopsThe animations loop invisibly, however long Codex grinds.
Yours to remapEvery state, image, and label is configurable in ~/.codex-rpc.json.
